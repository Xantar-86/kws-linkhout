# Foto's voor de Foto Pagina

In deze map plaats je de thumbnail afbeeldingen voor de foto albums op de website.

## Hoe werkt het?

1. Kies een representatieve foto uit je Google Foto's album
2. Sla deze op in deze map met het volgende naamgeving formaat:
   - Gebruik het album ID uit `src/app/fotos/page.tsx`
   - Bestandsnaam moet overeenkomen met het ID, bijvoorbeeld:
     - `herfst-voetbalkamp-2025.jpg`
     - `jeugdtornooi-2025.jpg`
     - `dribbelfestival-2016.jpg`

3. Het systeem zoekt automatisch naar een afbeelding met hetzelfde ID
4. Als er geen specifieke foto gevonden wordt, wordt een standaard ploegfoto gebruikt

## Tips

- Gebruik afbeeldingen van ongeveer 800x600 pixels (4:3 verhouding)
- Bestandsgrootte: probeer onder 200KB te blijven voor snelle laadtijden
- Formaat: JPG werkt het beste voor foto's
- Kies een foto die het evenement goed representeert
